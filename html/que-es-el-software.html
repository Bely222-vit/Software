¿Qué es el software?

Definición

Definición básica:
El software es el conjunto de programas que permiten que una computadora funcione y realice tareas.
No se puede tocar, pero controla todo lo que hace el equipo.

Ejemplos:

Sistema operativo, juegos, navegadores, aplicaciones.
